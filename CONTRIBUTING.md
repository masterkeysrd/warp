# Contributing to Warp

We welcome contributions to the Warp project!

## How to Contribute

1. **Fork the repository.**
2. **Create a new branch** for your feature or fix.
3. **Submit a Pull Request** describing your changes.

Please ensure that your code follows the existing style and that you add tests for any new functionality or bug fixes.
